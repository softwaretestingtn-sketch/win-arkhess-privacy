<p><strong>Dernière mise à jour :</strong> Avril 2026</p>

<p>Win Arkhess ("l'Application") est une application mobile gratuite de comparaison de prix dans les supermarchés en Tunisie. La protection de vos données personnelles est importante pour nous. Cette politique explique quelles données nous collectons, pourquoi, et comment vous pouvez les gérer.</p>

<h2>1. Données collectées</h2>

<h3>1.1 Données de compte</h3>
<p>Lorsque vous vous connectez avec Google, nous recevons :</p>
<ul>
  <li>Votre nom d'affichage</li>
  <li>Votre adresse email</li>
  <li>Votre photo de profil (URL Google)</li>
</ul>
<p>Si vous utilisez l'application sans compte (mode anonyme), un identifiant anonyme est généré automatiquement. Aucune donnée personnelle n'est collectée dans ce cas.</p>

<h3>1.2 Données de contribution</h3>
<p>Lorsque vous ajoutez un prix, nous enregistrons :</p>
<ul>
  <li>Le code-barres du produit</li>
  <li>Le prix renseigné</li>
  <li>L'enseigne sélectionnée</li>
  <li>La date de contribution</li>
  <li>Votre identifiant utilisateur (pour le classement et l'anti-triche)</li>
</ul>

<h3>1.3 Données d'utilisation</h3>
<ul>
  <li>Vos points et votre niveau de gamification</li>
  <li>Votre streak de contributions consécutives</li>
  <li>Vos défis journaliers et hebdomadaires</li>
  <li>Votre liste de courses (si vous utilisez cette fonctionnalité)</li>
</ul>

<h3>1.4 Données techniques</h3>
<ul>
  <li>Token de notification push (pour les alertes de prix)</li>
  <li>Type d'appareil et version de l'OS (collectés automatiquement par Expo)</li>
</ul>

<div class="highlight">
  <strong>Ce que nous ne collectons PAS :</strong> votre position GPS, vos contacts, vos messages, votre historique de navigation, ni aucune donnée sensible (santé, religion, orientation politique, etc.).
</div>

<h2>2. Utilisation des données</h2>
<p>Vos données sont utilisées exclusivement pour :</p>
<ul>
  <li>Afficher les prix comparés entre les supermarchés</li>
  <li>Gérer votre profil, vos points et votre classement</li>
  <li>Vous envoyer des notifications de baisse de prix (si activées)</li>
  <li>Lutter contre la triche et les contributions frauduleuses</li>
  <li>Améliorer l'application et ses fonctionnalités</li>
</ul>

<div class="highlight">
  <strong>Nous ne vendons jamais vos données personnelles à des tiers.</strong>
</div>

<h2>3. Stockage et sécurité</h2>
<p>Vos données sont stockées sur les serveurs de <strong>Google Firebase</strong> (Google Cloud Platform), situés en Europe. Firebase applique des mesures de sécurité de niveau entreprise, incluant le chiffrement en transit et au repos.</p>
<p>L'accès aux données est protégé par des règles de sécurité Firestore strictes : chaque utilisateur ne peut accéder qu'à ses propres données.</p>

<h2>4. Publicités</h2>
<p>L'application affiche des publicités via <strong>Google AdMob</strong>. AdMob peut collecter des données techniques (identifiant publicitaire, type d'appareil) pour personnaliser les publicités. Vous pouvez désactiver la personnalisation des annonces dans les paramètres de votre appareil Android (Paramètres → Google → Annonces).</p>
<p>Les publicités récompensées (vidéos) sont optionnelles et vous permettent de gagner des points en les regardant.</p>

<h2>5. Partage de données</h2>
<p>Vos données peuvent être partagées avec :</p>
<ul>
  <li><strong>Google Firebase</strong> — hébergement et authentification</li>
  <li><strong>Google AdMob</strong> — affichage de publicités</li>
  <li><strong>Expo</strong> — notifications push</li>
</ul>
<p>Aucun autre tiers n'a accès à vos données. Les contributions de prix sont visibles publiquement dans l'application (sans identification personnelle au-delà du nom d'affichage).</p>

<h2>6. Vos droits</h2>
<p>Conformément à la loi tunisienne n°2004-63 relative à la protection des données personnelles, vous avez le droit de :</p>
<ul>
  <li><strong>Accéder</strong> à vos données personnelles</li>
  <li><strong>Rectifier</strong> vos données inexactes</li>
  <li><strong>Supprimer</strong> votre compte et toutes vos données</li>
  <li><strong>Vous opposer</strong> au traitement de vos données</li>
</ul>

<h3>Comment supprimer votre compte :</h3>
<ol>
  <li>Ouvrez l'application Win Arkhess</li>
  <li>Allez dans l'onglet "Stats"</li>
  <li>Appuyez sur l'icône de déconnexion</li>
  <li>Pour une suppression complète, contactez-nous par email</li>
</ol>

<h2>7. Données des enfants</h2>
<p>L'application n'est pas destinée aux enfants de moins de 13 ans. Nous ne collectons pas sciemment de données personnelles d'enfants. Si vous êtes un parent et pensez que votre enfant nous a fourni des données, contactez-nous pour les supprimer.</p>

<h2>8. Modifications</h2>
<p>Cette politique peut être mise à jour. En cas de modification significative, une notification sera affichée dans l'application. La date de dernière mise à jour est indiquée en haut de cette page.</p>
